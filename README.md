# Neon Analog Clock

A modern cyberpunk-inspired analog clock built with HTML5 Canvas, CSS, and JavaScript.

## Features

- Neon glowing analog clock
- Smooth second hand animation
- Digital time display
- Responsive centered layout
- Pure HTML, CSS & JavaScript
- No external libraries

## Technologies

- HTML5
- CSS3
- JavaScript (Canvas API)

## Project Structure

Neon-Analog-Clock/
│── index.html
│── style.css
│── script.js
└── README.md

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or use VS Code Live Server.
